Informações
	O Connection é uma classe que está dentro da biblioteca MYSQL.
	Para se conectar no banco, o java precisa receber de onde, qual banco e qual usuário ele vai se conectar
	Para cada tabela no banco de dados vai ter uma classe DAO par aorganização do código
	O metodo cadastrar vai cadastrar um objeto da camada modelo (Objetos empacotados do modelo)
	Tudo será programado no construtor será executado quando se instancia um objeto Ex.: ClientesDAO dao = new ClienteDAO();
	Todo select realizado no banco retorna uma lista e essa lista deve ser guardada dentro do array no java
	Toda vez que usar o select tem que criar o objto do result set
	
	Os dados que são exibidos na tabela devem estar dentro de um DefaultTableModel quase igual o metodo listar que retorna uma lista
	Quando carrega o formulário ele preenche as informações
